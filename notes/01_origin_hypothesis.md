# 01 発想の由来

## 出発点

このノートの出発点は、現生人類だけが残った理由についての単純な説明への違和感である。

通俗的には、ホモ・サピエンスがネアンデルタール人やデニソワ人などの他集団を「倒した」「滅ぼした」と語られることがある。

しかし、その説明だけでは粗い。

より重要なのは、次の問いである。

> 小さな近縁集団が、大きな近縁集団と接触したとき、何が起こるか。

この問いを立てると、単純な勝敗ではなく、集団サイズ、交雑、希釈、疫病、環境変化、人口回復力の差が見えてくる。

## 単純な絶滅説ではなく、吸収仮説として見る

ここで考える仮説は、他集団が完全に消えたというより、独立した繁殖集団として維持できなくなった、というものである。

少数派集団は、多数派集団と接触したとき、次のような過程をたどりうる。

1. 多数派集団と接触する
2. 一部で交雑が起こる
3. 少数派由来の成分が混合集団に入る
4. 多数派集団内で希釈される
5. 不利な成分は選択によって減る
6. 有利または中立な成分だけが低頻度で残る
7. 少数派集団は独立した繁殖集団としては維持できなくなる

この場合、「消えた」のは個体や全成分ではなく、集団境界である。

## 絶滅・吸収・希釈の区別

このノートでは、以下を区別する。

| 概念 | 意味 |
|---|---|
| 絶滅 | 個体も集団も維持されず、成分も残らない状態 |
| 吸収 | 集団境界は消えるが、一部の成分は多数派に取り込まれる状態 |
| 希釈 | 少数派由来成分が、多数派集団内で世代ごとに薄まる状態 |
| 独立性喪失 | 集団としての境界や再生産単位が維持できなくなる状態 |

この区別が重要である。

「絶滅した」と見えるものの一部は、実際には「独立集団としては消えたが、一部の成分は吸収された」と表現した方がよい場合がある。

## 集団サイズの効果

少数派集団と多数派集団が交雑可能である場合、少数派の成分は多数派集団内で薄まりやすい。

例えば、少数派集団が1,000人、多数派集団が49,000人で接触した場合、混合後の少数派由来成分は単純には約2%になる。

このとき、少数派の成分は完全には消えない。

しかし、100%少数派として再生産される個体や集団は急速に減る。

ここから、次の構造が見える。

```text
少数派の個体数は減る
少数派由来の成分は残る
少数派集団としての独立性は失われる
```

## 疫学的脆弱性

小集団は、数が少ないだけでなく、疫病や環境変化に対して脆弱になりやすい。

理由は複数ある。

- 免疫関連の多様性が低くなりやすい
- 感染後の人口回復力が低い
- 成人死亡が労働力や出生率に直撃しやすい
- 配偶者探索が難しくなる
- 技術や文化の継承が途切れやすい
- 近親化によって有害な影響が出やすくなる

疫病は、集団の全員を殺す必要はない。

小集団では、一定割合の死亡や出生低下だけでも、集団維持の連鎖が崩れる可能性がある。

```text
感染
→ 成人死亡
→ 労働力・養育力の低下
→ 出生率低下
→ 子どもの死亡率上昇
→ 配偶者不足
→ 独立集団としての維持失敗
```

## この仮説から見える一般構造

ここで見えているのは、古人類だけの話ではない。

閉鎖的または小規模な集団は、内部ではまとまりを維持できるが、外部の大きな集団と接触したとき、次のような問題を持つ。

```text
隔離
→ 内部再生産
→ 多様性低下
→ 外部接続力の低下
→ 疫学的・人口学的脆弱性
→ 吸収・希釈・独立性喪失
```

この構造を、貴族内婚、疫学、企業組織へ転用する。

## 貴族内婚への接続

貴族は生物学的に別種ではない。

しかし、身分制度、財産、称号、婚姻制限によって、ある程度閉じた婚姻圏を作ることがある。

この場合、貴族階層も閉鎖集団として振る舞う。

外部と混ざると、法的・社会的な独立性は薄れるが、家系、文化、資産、ネットワークの一部は別の形で残る。

つまり、ここでも「完全消滅」ではなく「吸収と希釈」という見方が使える。

## 企業組織への接続

企業組織では、遺伝子ではなく、ルーティン、評価基準、暗黙知、意思決定様式が再生産される。

閉鎖的な組織では、内部で評価される行動が再生産される一方、外部から入った知識や人材が定着しないことがある。

ここでの問題は、外部知識がないことではない。

外部知識が内部で再生産されないことである。

この発想が、組織再生産モデルの中心になる。
