# 07 疫学と小集団脆弱性

このファイルでは、小集団が疫病や環境変化に対してなぜ脆弱になりうるのかを整理する。

ここでの目的は、古人類や貴族を医学的に単純化することではない。

目的は、閉鎖集団や小規模集団が、病原体、人口回復力、免疫多様性、再生産能力の面でどのような脆弱性を持ちうるかを整理し、その構造を企業組織の悪習感染へ接続することである。

## 大集団と小集団の違い

大集団は、感染が広がりやすい。

人の接触が多く、病原体が維持されやすいからである。

しかし、大集団は全滅しにくい。

一部が感染しても、未感染者、回復者、別地域の個体、別系統の集団が残る可能性がある。

一方、小集団は感染が入る頻度は低い場合があるが、一度入ると崩れやすい。

理由は、人口回復力が低く、成人死亡や出生率低下が集団維持に直撃しやすいからである。

## 疫病は全員を殺す必要がない

疫病が集団を崩すために、全員を殺す必要はない。

小集団では、一部の死亡だけでも次の連鎖が起こりうる。

```text
感染
→ 成人死亡
→ 狩猟・労働・養育力の低下
→ 出生率低下
→ 子どもの死亡率上昇
→ 配偶者不足
→ 技術・文化継承の失敗
→ 集団維持不能
```

つまり、疫病は直接的な全滅要因でなくても、集団崩壊の引き金になりうる。

## 免疫多様性

小集団では、免疫関連の多様性が低くなりやすい。

多様性が低い場合、特定の病原体に対して集団全体が似た脆弱性を持つ可能性がある。

これは、個々の個体が必ず弱いという意味ではない。

集団全体として、病原体に対する応答の幅が狭くなる可能性があるという意味である。

## 近親化との接続

小集団では、配偶者候補が限られやすい。

その結果、近親化が進む場合がある。

近親化は、劣性の有害な影響を表に出しやすくする可能性がある。

ただし、これは条件付きの話である。

```text
小集団
→ 配偶者候補の制限
→ 近親化
→ 有害な影響の顕在化
```

という経路はありうるが、必ず起こるわけではない。

栄養、環境、医療アクセス、出産条件、記録バイアスも同時に見る必要がある。

## 古人類への接続

古人類の小集団が、現生人類の大きな集団と接触した場合、問題は単なる競争ではない。

以下の複合が起こりうる。

- 人口規模の差
- 交雑可能性
- 少数派成分の希釈
- 病原体交換
- 免疫多様性の差
- 小集団の人口回復力の低さ
- 文化・技術継承の途切れ

この場合、古人類集団は完全に殺戮されたというより、疫学的・人口学的・交雑的な圧力の中で、独立集団として維持できなくなった可能性がある。

## 貴族への接続

貴族内婚でも、小集団脆弱性の構造が一部使える。

婚姻候補が狭くなり、内婚が長期化すると、特定家系で脆弱性が表に出やすくなる可能性がある。

ただし、貴族の場合は一般集団よりも栄養、住居、保護、医療アクセス、記録保存で有利な場合がある。

したがって、単純に「貴族は病弱」と言ってはいけない。

正確には、閉鎖的で小規模な婚姻圏が長く続いた場合、特定の医学的脆弱性や後継者問題が表に出やすくなる可能性がある、という限定的な見方にする。

## 企業組織への転用

企業における疫病は、物理的な病原体ではない。

企業における疫病に相当するものは、悪習、不正、隠蔽、責任回避、忖度、顧客軽視、異論排除などの有害ルーティンである。

これらは、社内ネットワークを通じて広がる。

```text
ある部署で隠蔽が許される
→ 近い部署が模倣する
→ 上司が黙認する
→ 報告しないことが生存戦略になる
→ 組織全体に広がる
```

この意味で、悪習は感染する。

## 組織の免疫

企業組織にも、免疫に相当する仕組みがある。

- 異論
- 内部通報
- 外部監査
- 顧客接触
- 外部人材
- 透明な評価制度
- 失敗を報告できる文化
- 権限の分散
- 部門間の人材移動

閉鎖的な組織では、これらの免疫が弱くなることがある。

その結果、有害ルーティンが広がりやすく、修正されにくくなる。

## 数理化への接続

悪習感染は、将来的には感染モデルとして扱える。

たとえば、以下の変数を考えられる。

| 変数 | 意味 |
|---|---|
| `S` | 悪習に感染していない構成員・部署 |
| `I` | 悪習を実行・黙認している構成員・部署 |
| `R` | 悪習から離脱・修正された構成員・部署 |
| `β` | 悪習感染率 |
| `γ` | 修正率 |
| `C` | 組織閉鎖性 |

閉鎖性が高いほど、悪習感染率 `β` は上がり、修正率 `γ` は下がる可能性がある。

```text
C が上がる
→ β が上がる
→ γ が下がる
→ 悪習が定着する
```

この段階では、数式を確定しない。

ただし、この方向により、悪習や不正を単なる道徳論ではなく、組織内の伝播過程として扱える。

## 最小命題

小集団や閉鎖集団は、外部からの衝撃に対して、単に個体数が少ないから弱いのではない。

免疫多様性、人口回復力、配偶者候補、修正能力、外部接続力が同時に制限されることで、脆弱化する。

企業組織では、この構造は悪習感染と修正能力の低下として現れる。
