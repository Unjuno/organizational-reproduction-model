# 03 企業組織への転用

このファイルでは、古人類・貴族内婚・疫学で見た閉鎖集団の構造を、企業組織へ転用する。

目的は、企業を生物と同一視することではない。

目的は、異なる領域に共通する「何が内部で再生産され、何が外部から入っても再生産されないか」という構造を見ることである。

## 対応関係

| 閉鎖集団モデル | 企業組織 |
|---|---|
| 遺伝子 | ルーティン・評価基準・暗黙知・意思決定様式 |
| 内婚 | 内部昇進・同質採用・派閥内登用 |
| 交雑 | 中途採用・M&A・外部提携・異業種人材の流入 |
| 不妊 | 外部知識が社内で再生産されない |
| 病原体 | 悪習・不正・隠蔽文化・責任回避 |
| 免疫 | 異論・監査・外部人材・顧客接触・透明性 |
| 近交弱勢 | 判断の同質化・革新低下・修正能力低下 |
| 吸収 | 買収・外部資本・人材流入による再編 |
| 希釈 | 独自文化や技術が広く薄く残る状態 |
| 消滅 | 倒産・事業撤退・組織空洞化・ブランドだけが残る状態 |

## 企業で再生産されるもの

企業で再生産されるのは、生物学的な遺伝子ではない。

企業で再生産されるのは、以下のようなものである。

- 評価基準
- 昇進基準
- 会議の作法
- 顧客への向き合い方
- 失敗の扱い方
- 異論の扱い方
- 責任の取り方
- 新しい知識の受け入れ方
- 社内で「有能」とされる行動様式

これらが、採用、育成、昇進、配置、評価、会議、稟議、報告制度を通じて次世代に渡される。

この意味で、企業組織にも「再生産」がある。

## 自然選択ではなく市場淘汰・組織淘汰・制度淘汰

企業組織にこのモデルを転用するとき、自然選択という語はそのまま使わない方がよい。

企業は生物ではなく、遺伝子を持つ個体でもない。

したがって、企業に起きるのは自然選択ではなく、より正確には以下である。

- **市場淘汰**: 顧客、市場、競合、技術変化によって、企業や事業が選別されること。
- **組織淘汰**: 組織内部の評価、昇進、配置、権限配分によって、人材やルーティンが選別されること。
- **制度淘汰**: 法制度、規制、資本市場、取引慣行、業界標準によって、組織形態や行動様式が選別されること。

この区別は重要である。

自然選択という言葉を使うと、企業の変化を生物学的過程と誤解しやすい。

このノートで扱うのは、自然選択ではなく、企業内外の制度・市場・評価システムが、どのルーティンを残し、どのルーティンを消すかという選別過程である。

```text
市場淘汰
→ 市場で通用しない事業が縮小・撤退する

組織淘汰
→ 社内で評価される人材・判断様式が昇進し、再生産される

制度淘汰
→ 規制、資本、慣行に適合しない組織形態が維持できなくなる
```

このため、企業組織への転用では、自然選択ではなく、市場淘汰・組織淘汰・制度淘汰として記述する。

## 中核命題

閉鎖的組織では、次の非対称性が生じる。

```text
有害ルーティンは内部で増える
外部知識は内部で増えない
```

この非対称性が、組織の病理を作る。

より正確には、次のように整理できる。

```text
内部で評価される行動は残る
外部環境に有効な知識は、残るとは限らない
```

閉鎖組織では、社内で生き残る能力と、外部環境で成果を出す能力がずれる。

```text
社内で有能
≠
市場で有効
```

このずれが、内部適応による外部不適応である。

## 組織内婚

組織内婚とは、内部昇進、同質採用、同じ価値基準の反復によって、似た人材や似た判断様式だけが再生産される状態である。

これは、血縁的な内婚ではない。

組織においては、誰が評価され、誰が昇進し、誰が次の意思決定者になるかによって、判断様式が再生産される。

組織内婚が強いと、以下が起きやすい。

- 内部事情に詳しい人が高く評価される
- 外部市場に詳しい人が軽視される
- 異論を出す人が昇進しにくい
- 中途人材が周辺化される
- 新しい判断基準が制度化されない
- 成功体験が過剰に再利用される

短期的には、組織内婚は効率を上げる。

共通言語があり、信頼関係があり、説明コストが低いからである。

しかし、長期的には、外部変化への感度を下げる。

## 吸収不妊

吸収不妊とは、外部人材、外部知識、M&A、提携、コンサルなどが入っても、それが組織内部で継続的に再生産されない状態である。

外部流入そのものは、吸収ではない。

外から入ったものが、次の意思決定、評価制度、製品、管理職登用、標準業務に残って初めて、吸収されたと言える。

吸収不妊の例は以下である。

- 中途人材が短期間で辞める
- 買収先の技術者が流出する
- 買収した技術が製品化されない
- 外部提携が社内制度に接続しない
- コンサル提案が資料だけで終わる
- 新規事業部が本体に潰される
- 外部出身者が昇進経路に乗らない

この場合、企業は外部知識を「入れている」が、再生産していない。

つまり、外部入力はあるが、組織内で子孫を残せていない。

## 悪習感染

悪習感染とは、有害なルーティンが社内ネットワークを通じて伝播する状態である。

悪習は、単に悪意ある個人がいるから広がるのではない。

それが社内で有利に働くと、悪習は生存戦略になる。

例として、以下がある。

- 問題を上に報告しない
- 数字をよく見せる
- 顧客より上司を優先する
- 責任を曖昧にする
- 失敗を隠す
- 異論を言う人を孤立させる
- 外部人材を「分かっていない」と排除する

このような行動が罰されず、むしろ評価される場合、悪習は感染する。

閉鎖性が高い組織では、外部監査、顧客接触、異論、透明性が弱くなりやすい。

そのため、悪習を止める「免疫」が働きにくくなる。

## M&A と交雑

M&A は、企業組織における交雑として見ることができる。

ただし、買収しただけでは交雑は成立しない。

重要なのは、買収先の技術、人材、判断様式、顧客理解が、買収元の内部で再生産されるかである。

失敗する M&A では、次のことが起きる。

```text
会社は買う
人材は流出する
技術は製品化されない
文化は本体に潰される
ブランドだけが残る
```

この状態は、外部要素の吸収不妊である。

一方、成功する M&A では、買収先の要素が本体の制度、製品、評価、意思決定に残る。

このとき、外部要素は組織内で再生産されたと言える。

## 中途採用と外部知識

中途採用も、単なる外部流入ではない。

重要なのは、中途人材が持ち込んだ知識や判断基準が、組織内で残るかである。

中途人材が個人として成果を出しても、その知識が制度化されなければ、組織は変わらない。

逆に、中途人材がいなくなっても、その人が持ち込んだ方法が標準業務や評価基準に残れば、外部知識は再生産されたと言える。

## 企業版の消滅

企業は生物のように単純に死ぬわけではない。

しかし、以下は組織的な消滅として扱える。

- 倒産
- 買収
- 事業撤退
- 中核人材の流出
- ブランドだけが残る
- 会社名は残るが意思決定能力が失われる
- 独自技術や独自文化が失われる
- 外部資本や親会社の制度に吸収される

このモデルでは、消滅を一種類にしない。

```text
完全消滅
吸収
希釈
独立性喪失
空洞化
```

に分けて見る。

## この転用で見えること

この転用によって、企業組織の問題を次のように読み替えられる。

| 企業現象 | 閉鎖集団モデルでの読み替え |
|---|---|
| 大企業病 | 組織内婚と内部適応の過剰 |
| 中途人材の定着失敗 | 吸収不妊 |
| M&A統合失敗 | 外部要素の再生産失敗 |
| 品質不正・隠蔽 | 悪習感染 |
| 新規事業が潰れる | 外部的ルーティンの不妊化 |
| 内部昇進偏重 | 組織内婚 |
| 市場変化への遅れ | 内部適応による外部不適応 |
| 買収後の空洞化 | 吸収または独立性喪失 |

## 注意点

この転用には注意が必要である。

企業は生物ではない。

貴族階層は別種ではない。

疫学の病原体と、組織内の悪習は同じものではない。

ここで比較しているのは、実体ではなく、以下の動態である。

```text
何が再生産されるか
何が再生産されないか
何が内部で増えるか
何が外部から入っても残らないか
どの時点で独立性を失うか
```

したがって、このモデルは比喩で終わらせるべきではないが、同時に生物学的説明を企業へそのまま移植するものでもない。

## 最小命題

企業組織への転用としての最小命題は、以下である。

> 組織は、外部知識を取り込めないから衰退するだけではない。外部知識が内部で再生産されず、有害ルーティンだけが内部で再生産されるとき、内部適応しながら外部不適応化する。
