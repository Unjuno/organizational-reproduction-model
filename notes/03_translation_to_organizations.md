# 03 企業組織への転用

## 対応関係

| 閉鎖集団モデル | 企業組織 |
|---|---|
| 遺伝子 | ルーティン・評価基準・暗黙知 |
| 内婚 | 内部昇進・同質採用 |
| 交雑 | 中途採用・M&A・外部提携 |
| 不妊 | 外部知識が社内で再生産されない |
| 病原体 | 悪習・不正・隠蔽文化 |
| 免疫 | 異論・監査・外部人材・顧客接触 |
| 吸収 | 買収・外部資本・人材流入による再編 |
| 消滅 | 倒産・事業撤退・組織空洞化 |

## 中核命題

閉鎖的組織では、次の非対称性が生じる。

```text
有害ルーティンは内部で増える
外部知識は内部で増えない
```

この非対称性が、組織の病理を作る。
