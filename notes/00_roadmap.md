# 00 ロードマップ

## このノート群の目的

このノート群では、閉鎖集団が内部再生産を通じて安定し、その一方で外部不適応・吸収不全・悪習感染を生む構造を整理する。

対象は企業組織に限らない。古人類集団、貴族内婚、疫学、小集団脆弱性を経由し、最終的に企業組織へ転用する。

## 基本姿勢

このモデルは、企業を生物と同一視するものではない。

比較するのは、次のような動態構造である。

```text
隔離
→ 内部再生産
→ 多様性低下
→ 外部接続力の低下
→ 吸収不全または希釈
→ 感染・衰退・消滅
```

## 読み順

### 1. 発想の由来

`01_origin_hypothesis.md`

現生人類だけが残った理由を、単純な殺戮説ではなく、集団サイズ、交雑可能性、少数派成分の希釈、小集団脆弱性、疫病などの複合として整理する。

### 2. 中核概念

`02_core_concepts.md`

閉鎖集団、内部再生産、吸収、希釈、組織内婚、吸収不妊、悪習感染など、このノート群で使う概念を定義する。

### 3. 企業組織への転用

`03_translation_to_organizations.md`

古人類・貴族・疫学の議論を、企業組織の閉鎖性、内部昇進、外部人材の定着失敗、M&A統合失敗、不祥事伝播へ写像する。

### 4. 既存理論との差分

`04_existing_theories.md`

組織慣性、探索と活用、吸収能力、組織ルーティン、大企業病、不祥事伝播などの既存概念との関係を整理する。

### 5. 未検証点と反証条件

`05_open_questions.md`

この枠組みがどこで破れるか、どう測定すべきか、どのような誤用リスクがあるかを整理する。

## 当面やらないこと

現段階では、以下は扱わない。

- 大規模シミュレーション
- 実証分析
- 企業診断ツール化
- 企業スコアリング
- 網羅的な文献レビュー
- 完成理論としての主張

まずは、概念の骨格と応用可能性を固定する。
