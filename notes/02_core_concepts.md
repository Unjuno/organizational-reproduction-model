# 02 中核概念

このファイルでは、以後のノートで使う概念を定義する。

ここでの用語は、生物学・社会史・疫学・組織論をそのまま同一視するものではない。異なる領域に共通して現れる「再生産の構造」を比較するための作業概念である。

## 閉鎖集団

外部との交配、婚姻、知識流入、人材流入、価値基準の交換が制限された集団。

例として、以下がある。

- 地理的に孤立した小集団
- 婚姻制限を持つ貴族階層
- 宗教的・身分的な内婚集団
- 同族企業
- 内部昇進偏重の企業
- 強い派閥文化を持つ組織

閉鎖集団は、短期的には安定しやすい。内部信頼、共通規範、意思決定の速さを得るからである。

しかし、長期的には外部との接続能力が落ち、多様性や修正能力を失う可能性がある。

## 内部再生産

集団内部の要素だけで、次世代の構造を作ること。

再生産されるものは領域によって異なる。

| 領域 | 再生産されるもの |
|---|---|
| 生物集団 | 遺伝子 |
| 貴族階層 | 家系、称号、婚姻規則、財産ネットワーク |
| 疫学 | 病原体、免疫記憶、感染経路 |
| 企業組織 | ルーティン、評価基準、暗黙知、意思決定様式 |

このモデルで重要なのは、何が内部で残り、何が内部で残らないかである。

## 吸収

少数派集団が多数派集団に取り込まれ、独立した集団境界を失うこと。

吸収は、完全消滅とは異なる。

吸収された集団の一部の成分は、多数派集団の中に残ることがある。

例として、次のような状態を考える。

```text
独立した少数派集団は消える
しかし、少数派由来の成分は多数派集団内に残る
```

企業組織では、買収や統合によって企業名や部門は消えても、技術、人材、文化、顧客基盤の一部が残る場合がある。

## 希釈

少数派由来の成分が、多数派集団内で世代や再編を経るごとに薄く広がること。

単純化すると、外部集団と混ざることで以下のように比率が下がる。

```text
100%
→ 50%
→ 25%
→ 12.5%
```

希釈は、消滅とは違う。

100%の状態は減るが、少数派由来成分を一部持つ個体や組織要素は広がることがある。

## 独立性喪失

個体や成分が完全に消えるのではなく、集団としての境界や自己再生産能力を失うこと。

このモデルでは、消滅を以下のように細かく分ける。

| 状態 | 意味 |
|---|---|
| 完全消滅 | 成分も境界も残らない |
| 吸収 | 境界は消えるが成分は残る |
| 希釈 | 成分は残るが薄く広がる |
| 独立性喪失 | 自分自身だけでは再生産できなくなる |

企業でいえば、会社名は残っていても、独自技術、人材、判断基準、顧客基盤が失われていれば、実質的には独立性を失っている場合がある。

## 組織内婚

内部昇進、同質採用、同じ部署・同じ学歴・同じ価値基準の反復によって、似た人材や似た判断様式だけが再生産される状態。

これは生物学的な内婚そのものではない。

組織における「誰が誰を評価し、誰が次の管理職や意思決定者になるか」という再生産の閉鎖性を指す。

組織内婚が強くなると、次のようなことが起きやすい。

- 社内政治への適応が評価される
- 外部市場への適応が軽視される
- 異論を持つ人材が昇進しにくい
- 中途人材が定着しにくい
- 新しい判断様式が制度化されにくい

## 吸収不妊

外部人材、M&A、提携、コンサル、外部知識が入っても、それが組織内部で標準的なルーティンとして再生産されない状態。

ここで重要なのは、外部流入そのものではない。

重要なのは、流入したものが組織内で残り、次の意思決定や制度に組み込まれるかである。

例として、以下がある。

- 中途人材が短期間で辞める
- M&A先の技術者が流出する
- 買収した技術が製品化されない
- コンサル提案が実行されない
- 新規事業部が本体に潰される
- 外部出身者が昇進経路に乗らない

吸収不妊が強い組織では、外部から良いものを入れても、内部では増えない。

## 悪習感染

隠蔽、忖度、責任回避、顧客軽視、数字の粉飾、異論排除などの有害ルーティンが、社内ネットワークを通じて広がる状態。

悪習は、単なる個人の性格ではない。

周囲がそう振る舞い、それが評価され、罰されず、むしろ生存戦略として有利になると、悪習は組織内で感染する。

例として、以下がある。

- 問題を報告しない
- 顧客より上司を優先する
- 数字をよく見せる
- 失敗を隠す
- 異論を出す人を孤立させる
- 責任の所在を曖昧にする

悪習感染は、閉鎖性が高く、外部監査や異論が弱い組織で起きやすい。

## 内部適応による外部不適応

社内では高く評価される行動が、市場、顧客、技術変化、社会的信頼に対しては不適応になる状態。

これはこのモデルの中心概念である。

閉鎖組織では、内部で生き残る能力と、外部環境で成果を出す能力がずれることがある。

```text
社内で有能
≠
外部環境で有効
```

このずれが大きくなると、組織は内部的には安定して見えるが、外部変化に対して急に弱くなる。

## 閉鎖集団の自己希釈

閉鎖集団が外部と混ざることで、独立性は失うが、一部の要素は広く薄く残る状態。

これは、古人類の吸収仮説、貴族階層の外婚、企業買収や統合に共通する構造である。

自己希釈は、単なる敗北ではない。

独立性は失われるが、成分の一部は残る。

このモデルでは、その状態を「消えた」とだけ見ず、「何が消え、何が残ったのか」に分けて見る。

## 最小命題

このノート群の中核命題は、以下である。

> 閉鎖集団は、内部でうまく再生産されるものと、外部から入っても再生産されないものの差によって、安定し、脆弱化し、希釈され、あるいは消滅する。

企業組織に置き換えると、以下になる。

> 組織は、有害ルーティンを内部で再生産し、外部知識を再生産できなくなったとき、内部適応しながら外部不適応化する。
