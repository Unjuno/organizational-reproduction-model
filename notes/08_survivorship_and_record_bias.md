# 08 生存者バイアスと記録バイアス

このファイルでは、閉鎖集団を見るときに生じる生存者バイアスと記録バイアスを整理する。

この論点は、古人類、貴族、企業のすべてに関係する。

見えている集団は、実際に存在した集団全体ではない。

残ったもの、記録されたもの、名前が残ったもの、成功したものだけを見ている可能性がある。

## なぜバイアスが重要か

閉鎖集団を分析するとき、以下のような誤りが起きやすい。

```text
記録に残ったもの
=
実際に存在したもの全体
```

しかし、これは成り立たない。

記録に残る確率は、集団の地位、財産、政治的成功、文化的威信、保存制度、後世の関心によって偏る。

そのため、見えているデータだけを使うと、閉鎖集団の実態を過大評価または過小評価する可能性がある。

## 貴族における記録バイアス

貴族の場合、記録に残りやすいのは以下である。

- 王家
- 大貴族
- 成功した家系
- 正嫡系統
- 男系系譜
- 大土地所有者
- 政治的に重要だった家系
- 戦争や婚姻で大きな役割を持った家系

一方で、見えにくいのは以下である。

- 下級貴族
- 没落した家系
- 断絶した家系
- 庶流
- 非嫡出子
- 女系で吸収された家系
- 外婚によって階層境界を失った家系
- 記録を保存できなかった家系

このため、記録に残った貴族像だけを見ると、貴族全体を誤って理解する可能性がある。

## 生存者バイアス

生存者バイアスとは、残ったものだけを見て、消えたものを見落とすことである。

貴族階層を見る場合、記録に残った家系は、そもそも生き残る条件を満たした家系である。

したがって、それらを見て「貴族は長く続いた」と言うだけでは不十分である。

本来は、続かなかった家系、吸収された家系、外婚で希釈された家系も含めて見る必要がある。

```text
観察される貴族家系
=
存在した貴族家系の一部
```

## 病弱化を見るときの注意

貴族の病弱化や近親化を見るときも、記録バイアスが問題になる。

病気や奇形や後継者問題が記録に残りやすい家系もあれば、記録が残らないまま消えた家系もある。

また、王家や大貴族は記録が多いため、医学的問題が見えやすい。

その一方で、一般の下級貴族や周辺家系の情報は不足しやすい。

したがって、特定の有名家系だけを見て「貴族全体」を語ってはいけない。

## 古人類におけるバイアス

古人類でも、見えている証拠には偏りがある。

化石や遺物として残るものは、保存条件、発掘条件、研究関心に強く左右される。

したがって、ある集団が見えないからといって、存在しなかったとは限らない。

また、独立集団としては消えても、遺伝的成分や文化的要素が別の集団に吸収されて残った可能性がある。

ここでも重要なのは、以下の区別である。

```text
記録に残らない
≠
存在しなかった

独立集団として見えない
≠
成分が完全に消えた
```

## 企業におけるバイアス

企業組織でも同じ問題がある。

記録に残るのは、成功企業、上場企業、大企業、有名な失敗事例、報道された不祥事に偏りやすい。

一方で、以下は見えにくい。

- 静かに消えた中小企業
- 買収後に名前だけ消えた企業
- 事業部として吸収された組織
- 中途人材が辞めたが記録されない事例
- 小規模なM&A失敗
- 表に出ない不正や隠蔽
- 外部知識が制度化されなかった事例

このため、企業研究でも、観察できる事例だけで一般化すると危険である。

## 社会的カテゴリーと記録

貴族のような社会的カテゴリーは、記録によって固定される。

称号、家系図、婚姻記録、土地記録、宮廷記録、戦争記録などが残ることで、後世から見た「貴族像」が作られる。

しかし、その記録は中立ではない。

正統性を示すための記録、財産権を守るための記録、政治的地位を示すための記録が多い。

そのため、記録に残った貴族像は、制度が見せたかった貴族像でもある。

## 社会的カテゴリーと遺伝的偏りの両立

生存者バイアスや記録バイアスを考えるとき、貴族をどう扱うかが重要になる。

貴族は、生物学的な別種ではない。

第一義的には、法的・社会的・政治的カテゴリーである。

しかし、社会的カテゴリーが婚姻を制限すると、結果として家系的・遺伝的偏りが生じることがある。

```text
社会的カテゴリー
→ 婚姻制限
→ 記録された正統性
→ 閉鎖的婚姻圏
→ 家系的・遺伝的偏り
```

このため、以下の二つは両立する。

```text
貴族は社会的カテゴリーである
貴族は婚姻隔離を通じて家系的偏りを持ちうる
```

この区別をしないと、二つの誤りが起きる。

一つは、貴族を生物学的に別集団のように扱う誤りである。

もう一つは、社会的カテゴリーだから集団構造は一切生じないとみなす誤りである。

このノートでは、その中間を扱う。

## このモデルへの影響

生存者バイアスと記録バイアスは、組織再生産モデル全体に影響する。

なぜなら、このモデルは「何が残るか」を扱うからである。

残ったものを見るときには、以下を分ける必要がある。

| 問い | 内容 |
|---|---|
| 何が実際に存在したのか | 母集団の問題 |
| 何が生き残ったのか | 生存者バイアス |
| 何が記録されたのか | 記録バイアス |
| 何が制度的に正統とされたのか | 権力による選別 |
| 何が後世に研究されたのか | 研究関心の偏り |

したがって、このモデルでは、単に「残ったもの」を見るだけでは不十分である。

「なぜそれが残って見えているのか」も見る必要がある。

## 最小命題

閉鎖集団を分析するとき、観察されるものは、実在したもの全体ではない。

残ったもの、記録されたもの、正統とされたもの、研究対象になったものだけが見えている可能性がある。

このため、吸収、希釈、消滅、病弱化、企業衰退を扱うときには、生存者バイアスと記録バイアスを明示的に入れる必要がある。
