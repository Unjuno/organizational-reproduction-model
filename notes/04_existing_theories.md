# 04 既存理論との差分

このファイルでは、組織再生産モデルが既存の組織論とどこで重なり、どこで異なるのかを整理する。

前提として、このノートは既存理論を否定するものではない。

むしろ、既存理論で個別に扱われてきた現象を、閉鎖集団の再生産構造として接続し直すことを目的とする。

## 近い既存概念

このモデルに近い既存概念には、以下がある。

- 組織慣性
- 構造的慣性
- 組織生態学
- 探索と活用
- 吸収能力
- 組織ルーティン
- 内部労働市場
- 大企業病
- M&A後の統合失敗
- 不祥事の伝播
- 社会的感染
- 同族経営
- 派閥・学閥・内部昇進

これらは、すでに組織の閉鎖性、学習、変化への抵抗、外部知識の取り込み、悪い慣習の拡散を説明してきた。

このノートの目的は、それらの概念を置き換えることではない。

## 既存理論との重なり

### 組織慣性

組織慣性は、組織が一度作った構造やルーティンを変えにくいことを説明する。

このモデルでは、組織慣性を「内部再生産が強すぎる状態」として読む。

つまり、変われないのは単に意思決定が遅いからではなく、同じ評価基準、同じ人材経路、同じ成功体験が繰り返し再生産されるからである。

### 探索と活用

探索と活用は、新しい知識や市場を探る活動と、既存能力を利用する活動のバランスを扱う。

このモデルでは、活用が強くなりすぎる状態を、内部再生産の強化として見る。

既存ルーティンの活用は短期的には成果を出す。

しかし、外部環境が変わると、活用され続けた内部ルーティンが外部不適応を生む。

### 吸収能力

吸収能力は、外部知識の価値を認識し、取り込み、利用する能力を指す。

このモデルでは、吸収能力を「外部知識が組織内で再生産される条件」として読む。

外部知識を入れることと、外部知識が内部で増えることは違う。

この差を、ここでは吸収不妊として整理する。

### 組織ルーティン

組織ルーティンは、組織内で反復される行動様式や判断手順である。

このモデルでは、ルーティンを企業における再生産単位として扱う。

ただし、ルーティンを遺伝子そのものとみなすわけではない。

重要なのは、ルーティンが採用、昇進、評価、会議、報告制度を通じて次世代に残ることである。

### 内部労働市場

内部労働市場は、企業内部で人材が採用・育成・昇進していく仕組みである。

このモデルでは、内部労働市場を組織内婚の制度的基盤として見る。

内部労働市場は、短期的には安定性と忠誠心を作る。

しかし、閉鎖性が強くなると、外部人材や外部知識が昇進経路に乗らず、吸収不妊が起きる。

### 不祥事の伝播

不祥事や隠蔽は、個人の逸脱としてだけでなく、組織内で学習・模倣・黙認されるルーティンとして広がることがある。

このモデルでは、それを悪習感染として扱う。

悪習は、罰されず、むしろ短期的に有利になると、組織内で再生産される。

## このモデルの差分

差分は、個別概念の新規性ではない。

差分は、既存概念を以下の一つの流れに接続する点にある。

```text
隔離
→ 内部再生産
→ 多様性低下
→ 吸収不全
→ 有害ルーティンの固定・感染
→ 外部不適応
→ 衰退・吸収・消滅
```

既存組織論では、以下が別々に語られやすい。

| 既存概念 | よく扱う問題 |
|---|---|
| 組織慣性 | 変化しにくさ |
| 探索と活用 | 新規探索と既存利用の配分 |
| 吸収能力 | 外部知識を取り込む能力 |
| 組織ルーティン | 反復される行動様式 |
| 内部労働市場 | 内部昇進と人材形成 |
| 不祥事伝播 | 悪い行動の拡散 |

このノートでは、それらを次の問いに接続する。

> 閉鎖集団では、何が内部で再生産され、何が外部から入っても再生産されないのか。

この問いによって、組織慣性、吸収能力不足、M&A失敗、不祥事伝播、大企業病を、別々の症状ではなく、同じ閉鎖集団構造の別の現れとして見る。

## 観察帰納ではなく、生成機構から見る

既存の組織論は、多くの場合、企業で観察された現象から概念を作る。

例えば、企業が変われない、外部知識を吸収できない、不祥事が広がる、M&Aが失敗する、といった観察から理論化する。

このノートのアプローチは少し違う。

ここでは、古人類、貴族内婚、疫学、小集団脆弱性に共通する構造を先に見る。

そのうえで、同じ構造が企業組織でも現れるかを考える。

つまり、出発点は企業観察だけではなく、閉鎖集団一般の生成機構である。

```text
閉鎖集団は何を内部で再生産するか
外部と接触したとき何が吸収され、何が不妊化するか
有害なものはなぜ内部で増えるか
独立性はどのように失われるか
```

この問いから企業組織を見ることが、このノートの独自性である。

## 新規性の置き方

このノートで主張できる新規性は、以下である。

```text
単一概念の発明ではなく、複数領域の接続の仕方に新規性がある。
```

より具体的には、以下を接続する点にある。

- 古人類の吸収・希釈
- 貴族内婚と家系の維持失敗
- 小集団の疫学的脆弱性
- 企業の内部昇進と同質化
- 外部知識の再生産失敗
- 悪習の感染
- 組織の衰退・吸収・空洞化

この接続によって、組織論の個別概念を閉鎖集団の再生産構造として再配置する。

## このモデルで言えること

このモデルは、以下のような問いを立てるために使える。

- なぜ外部人材を入れても組織が変わらないのか
- なぜM&Aで技術や人材が残らないのか
- なぜ悪い慣習だけが社内で広がるのか
- なぜ内部では有能な人が外部環境では不適応になるのか
- なぜ閉鎖的組織は安定して見えた後に急に弱くなるのか
- なぜ組織は完全に消えるのではなく、吸収・希釈・空洞化するのか

## このモデルで言いすぎてはいけないこと

以下は、このノートでは主張しない。

- 既存組織論は間違っている
- 企業は生物と同じである
- 閉鎖性は常に悪である
- 内部昇進は常に悪である
- 外部人材を入れれば必ず良くなる
- 悪習はすべて感染モデルで説明できる
- 企業衰退はすべて閉鎖性で説明できる

閉鎖性は、短期的には安定、信頼、効率を生む。

問題は、環境変化が起きたときに、閉鎖性が外部知識の再生産を妨げ、有害ルーティンだけを残す場合である。

## 現段階の位置づけ

現段階では、これは完成理論ではない。

文献レビューでも、実証分析でも、シミュレーションでもない。

まずは、ここで議論した閉鎖集団の再生産構造を、既存組織論と接続するための概念整理である。

次に必要なのは、以下である。

- どの既存理論とどこまで重なるかを整理すること
- どの現象に最も適用しやすいかを絞ること
- 反証条件を明確にすること
- 誤用リスクを抑えること
