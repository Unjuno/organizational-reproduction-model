# 06 貴族内婚と家系の希釈

このファイルでは、古人類の吸収仮説と企業組織への転用の中間にある事例として、貴族内婚を整理する。

ここで扱う貴族は、生物学的な別集団ではない。

貴族は第一義的には、法的・社会的・政治的カテゴリーである。

ただし、婚姻制限、称号、財産、相続、正統性の維持によって、長期的には閉じた婚姻圏を作ることがある。

この意味で、貴族は「社会的カテゴリーでありながら、婚姻隔離を通じて遺伝的・家系的サブ構造を持ちうる集団」として扱える。

## なぜ内婚するのか

貴族内婚には、短期的な合理性がある。

- 財産を守る
- 称号を守る
- 家名を守る
- 政治同盟を作る
- 正統な相続を維持する
- 外部への資産流出を防ぐ
- 婚姻を通じて同階層のネットワークを維持する

この段階では、内婚は単なる不合理ではない。

むしろ、家系と財産を維持するための制度的な合理性を持つ。

## 合理性が拘束に変わる

問題は、短期的に合理的だった制度が、長期的には拘束になる点である。

婚姻相手を同階層に限定すると、候補者の範囲が狭くなる。

候補者の範囲が狭くなると、近親化、後継者不足、相続争い、婚姻戦略の硬直化が起きやすくなる。

つまり、内婚は初期には家系を守るが、長期には家系の再生産可能性を弱める場合がある。

```text
財産を守るための内婚
→ 婚姻候補の縮小
→ 近親化・後継者不足
→ 家系維持の困難化
→ 外婚・吸収・希釈
```

## 希釈

閉鎖集団が外部と混ざると、内部成分は急速に薄まる。

単純化すると、閉鎖集団の成分を100%持つ個体が外部集団と子を持つ場合、次のように比率は下がる。

```text
100%
→ 50%
→ 25%
→ 12.5%
→ 6.25%
```

ここで重要なのは、100%の状態は減るが、内部由来成分を少し持つ個体は広がりうることである。

したがって、貴族階層の消滅は、完全消滅ではなく、吸収と希釈として見ることができる。

## 貴族は「消えた」のか

貴族が消えると言う場合、何が消えたのかを分ける必要がある。

| 消えるもの | 残りうるもの |
|---|---|
| 法的特権 | 家名 |
| 身分制度 | 資産 |
| 閉鎖的婚姻圏 | ネットワーク |
| 政治的支配階層 | 文化的威信 |
| 純粋な階層境界 | 家系由来成分 |

このため、「貴族は倒された」とだけ見るのは粗い。

革命、法制度変更、課税、戦争、産業化、民主化などの外圧は重要である。

しかし同時に、内婚による閉鎖性そのものが、家系の維持可能性を下げた可能性もある。

## 病弱化と近交弱勢

小規模で内婚的な集団では、近親交配係数が上がりやすい。

その結果、劣性の有害な影響が表に出やすくなる可能性がある。

ただし、ここでは次の点を区別する必要がある。

```text
少数集団だから必ず病弱になる
```

という主張は強すぎる。

より正確には、以下の条件が重なった場合に脆弱性が高まる。

- 小規模である
- 婚姻候補が狭い
- 内婚が長く続く
- 有害変異が蓄積またはホモ接合化する
- 外部からの遺伝的流入が少ない
- 後継者数が少ない
- 記録に残る家系が偏っている

また、貴族は栄養、住居、医療アクセス、保護、記録保存では一般集団より有利な場合もある。

したがって、「貴族全体が病弱」と断定してはいけない。

正しくは、閉鎖的で小規模な婚姻圏が長く続くと、特定家系で医学的脆弱性が表に出やすくなる可能性がある、という限定的な主張にする。

## 具体例の扱い

ハプスブルク家のように、長期内婚の例として言及される家系はある。

ただし、このノートでは現段階で具体例を断定的に扱わない。

具体例を入れる場合は、系譜、婚姻関係、近親交配係数、健康状態、後継者問題、同時代の政治条件を分けて確認する必要がある。

現段階では、具体例は文献確認後に追加する候補として扱う。

## 青い血

「青い血」は、実際に血液が青いという意味ではない。

このノートでは、貴族性が身体的特徴として記号化される例として扱う。

典型的には、屋外労働が少ないことにより日焼けが少なく、肌が明るく見え、静脈が青く見えることが、高貴さの記号として解釈される、という構造である。

```text
屋外労働が少ない
→ 日焼けが少ない
→ 静脈が目立つ
→ 青い血という記号になる
```

この話の重要点は、身体的特徴そのものではなく、社会的地位が身体的記号として読まれることである。

## 社会的カテゴリーと遺伝的偏り

貴族は、第一義的には社会的カテゴリーである。

しかし、社会的カテゴリーであることと、婚姻隔離によって遺伝的・家系的偏りが生じることは矛盾しない。

```text
社会的カテゴリー
→ 婚姻制限
→ 遺伝子流動の制限
→ 家系的・形質的偏り
```

この順番で見る必要がある。

貴族を生物学的に別種として扱ってはいけない。

一方で、長期の婚姻隔離が何の集団構造も作らないと考えるのも不十分である。

この中間を扱うことが、このノートでの貴族内婚の位置づけである。

## 古人類・企業との接続

古人類では、少数派集団が多数派に吸収され、独立した繁殖集団としての境界を失う可能性を考えた。

貴族では、閉鎖的婚姻圏が外婚・制度変化・近代化によって希釈され、階層としての境界を失う可能性を考える。

企業では、閉鎖的な内部昇進・同質採用が外部知識を吸収できず、やがて買収・撤退・空洞化によって独立性を失う可能性を考える。

三者に共通するのは、以下である。

```text
閉鎖
→ 内部再生産
→ 短期安定
→ 多様性低下
→ 外部接続力低下
→ 吸収・希釈・独立性喪失
```
