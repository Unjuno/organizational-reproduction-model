# 05 未検証点と反証条件

このファイルでは、この枠組みの未検証点、反証条件、測定課題、誤用リスクを整理する。

重要なのは、このモデルが単なる比喩では終わらず、将来的に数理化できる点である。

現段階では数式モデルやシミュレーションを作らないが、閉鎖性、内部再生産、吸収不全、悪習感染、独立性喪失を変数として扱えるように整理する。

## この枠組みでできること

この枠組みでできることは、閉鎖集団の現象を数理化可能な形に置き直すことである。

通常の組織論では、以下のような言葉が使われる。

- 閉鎖的な組織
- 大企業病
- 外部人材がなじまない
- 悪い文化がある
- 変化に弱い
- M&Aがうまくいかない

これらは有用な観察だが、そのままでは測定しにくい。

このモデルでは、それらを次のように変数化する方向を持つ。

| 言葉 | 数理化する場合の変数候補 |
|---|---|
| 閉鎖的な組織 | 閉鎖性 `C` |
| 内部昇進・同質採用 | 組織内婚度 `F` |
| 多様性の低下 | 認知多様性 `H` |
| 外部人材が残らない | 外部知識再生産率 `B` |
| 外部知識を使えない | 吸収能力 `A` |
| 悪い文化が広がる | 悪習感染率 `β` |
| 悪習が修正される | 修正率 `γ` |
| 変化に弱い | 外部変化率 `T` と外部適応度 `E` |
| 衰退・買収・撤退 | 組織死亡ハザード `Z` |

つまり、このモデルの価値は、「観察語」を「状態変数」に変換できる点にある。

## 数理化の最小方向

将来的には、以下のような最小関係を置ける。

```text
閉鎖性 C が上がる
→ 外部知識再生産率 B が下がる
→ 認知多様性 H が下がる
→ 外部適応度 E が下がる
→ 組織死亡ハザード Z が上がる
```

また、悪習感染については、以下のように見ることができる。

```text
閉鎖性 C が上がる
→ 悪習感染率 β が上がる
→ 修正率 γ が下がる
→ 悪習が組織内で定着する
```

この段階では数式そのものを確定しない。

ただし、後で数式化できるように、概念を変数候補として整理しておく。

## 未検証点

この枠組みには、まだ未検証の点が多い。

- 組織閉鎖性をどう測るか
- 組織内婚をどう測るか
- 外部知識の再生産をどう測るか
- 悪習感染をどう測るか
- 悪習の修正率をどう測るか
- 独立性喪失をどう判定するか
- 既存組織論との差分をどこまで明確化できるか
- 古人類、貴族、疫学、企業の対応がどこで破れるか

## 測定課題

このモデルを実証可能にするには、概念を代理変数に落とす必要がある。

例として、以下が考えられる。

| 概念 | 測定候補 |
|---|---|
| 閉鎖性 | 中途管理職比率の低さ、外部役員比率の低さ、社内用語依存、意思決定階層の深さ |
| 組織内婚 | 内部昇進率、同一部署出身者の昇進集中、同一学歴・同一経歴の集中 |
| 吸収不妊 | 中途人材の離職率、M&A後の人材流出、外部由来制度の定着率 |
| 悪習感染 | 不正・品質事故・内部通報・監査指摘の部門間拡散 |
| 修正率 | 指摘後の改善速度、再発率、監査対応の実効性 |
| 外部適応 | 新規顧客比率、新製品売上比率、市場シェア変化、技術転換成功率 |
| 独立性喪失 | 買収、事業撤退、主要人材流出、ブランドだけが残る状態 |

ただし、これらはまだ仮の測定候補である。

現段階では、測定方法を固定しない。

## 反証条件

この枠組みは、以下が確認される場合に弱くなる。

- 閉鎖性が高くても外部適応が高い
- 外部知識が再生産されなくても組織成果が落ちない
- 有害ルーティンが感染しない
- 同質性がむしろ長期的な革新を支えている
- 内部昇進偏重でも外部人材や外部知識が十分に制度化される
- 衰退が閉鎖性ではなく、規制・景気・資本制約だけで説明できる
- M&Aや中途採用の失敗が、組織側ではなく外部人材側の選抜問題だけで説明できる

## 誤用リスク

この枠組みには誤用リスクがある。

特に、以下は避ける必要がある。

- 企業を生物と同一視すること
- 貴族や小集団を単純に「病弱」と決めつけること
- 遺伝的説明を社会的差別へ転用すること
- 閉鎖性を常に悪とみなすこと
- 内部昇進を常に悪とみなすこと
- 外部人材を入れれば必ず改善すると考えること
- 企業衰退をすべて閉鎖性で説明すること

閉鎖性は、短期的には安定、信頼、効率を生むことがある。

問題は、外部環境が変化したときに、閉鎖性が外部知識の再生産を妨げ、有害ルーティンだけを残す場合である。

## 当面やらないこと

現段階では、以下は行わない。

- 大規模シミュレーション
- 実証分析
- 企業診断スコア化
- 網羅的な文献レビュー
- 事例研究の断定
- 完成理論としての主張

まずは、概念を数理化可能な形に整理する。

## 次の課題

次の課題は、どの現象にこのモデルを最初に適用するかを絞ることである。

候補は以下である。

- M&A後の統合失敗
- 中途人材の定着失敗
- 品質不正や隠蔽の伝播
- 新規事業が本体に潰される現象
- 同族企業の後継者問題
- 内部昇進偏重企業の外部不適応

最初は、測定しやすい現象から始めるべきである。
