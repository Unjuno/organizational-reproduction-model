# 組織再生産モデル

このリポジトリは、古人類集団、貴族内婚、疫学、企業組織に共通する「隔離・内部再生産・吸収不全・希釈・感染・消滅」の構造を整理する日本語概念ノートです。

企業を生物と同一視するものではありません。異なる領域に見える現象の背後にある、閉鎖集団の動態構造を比較するための作業ノートです。

## 概要

閉鎖集団は、内部で安定的に再生産されるものと、外部から入っても再生産されないものの差によって、安定し、脆弱化し、希釈され、あるいは消滅します。

このノートでは、その構造を以下の領域をまたいで整理します。

- 古人類集団の吸収仮説
- 貴族内婚と家系の希釈
- 小集団の疫学的脆弱性
- 生存者バイアスと記録バイアス
- 企業組織における閉鎖性と外部不適応

## 発想の流れ

```text
古人類の吸収仮説
→ 貴族内婚と家系の希釈
→ 小集団の疫学的脆弱性
→ 生存者バイアスと記録バイアス
→ 企業組織への転用
→ 組織再生産モデル
```

出発点は、現生人類だけが残った理由を「他集団を一方的に倒したから」と見るのではなく、集団サイズ、交雑可能性、少数派成分の希釈、病原体交換、小集団脆弱性の複合として考える仮説です。

そこから、閉鎖集団がどのように内部再生産を強め、外部との接続能力を失い、最終的に吸収・希釈・衰退するのかを、貴族内婚、疫学、企業組織へ拡張して考えます。

## このモデルで見るもの

このモデルが見るのは、単なる閉鎖性ではありません。

重要なのは、次の非対称性です。

```text
内部の有害ルーティンは再生産される
外部から入った知識や人材は再生産されない
```

企業組織に置き換えると、内部ではうまく適応した行動が、外部環境に対しては不適応になる状態を扱います。

また、このモデルの狙いは、閉鎖性・吸収不全・悪習感染といった観察語を、将来的に状態変数として扱える形へ整理することでもあります。

## 中核概念

- **閉鎖集団**: 外部との交配、婚姻、知識流入、人材流入が制限された集団。
- **内部再生産**: 集団内部の要素だけで次世代の構造を作ること。
- **吸収**: 少数派集団が多数派に取り込まれ、独立集団としては消えること。
- **希釈**: 少数派由来成分が、多数派集団内で薄く広がること。
- **組織内婚**: 内部昇進、同質採用、同じ評価基準の反復によって、似た人材・判断様式だけが再生産される状態。
- **吸収不妊**: 外部知識や外部人材が入っても、組織内部で継続的に再生産されない状態。
- **悪習感染**: 隠蔽、忖度、責任回避、顧客軽視、異論排除などの有害ルーティンが社内で伝播する状態。
- **内部適応による外部不適応**: 社内では高く評価される行動が、市場・顧客・技術変化に対しては不適応になる状態。

## 既存組織論との差分

このノートは、既存の組織論を否定するものではありません。

組織慣性、探索と活用、吸収能力、組織ルーティン、大企業病、内部労働市場、M&A統合失敗、不祥事伝播などの既存概念を、隔離・再生産・吸収不全・感染という共通構造から整理します。

新規性は、単一概念の発明ではなく、複数領域の現象を閉鎖集団の再生産構造として接続する点にあります。
