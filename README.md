# 組織再生産モデル

このリポジトリは、古人類集団、貴族内婚、疫学、企業組織に共通する「隔離・内部再生産・吸収不全・希釈・感染・消滅」の構造を整理する日本語概念ノートです。

## 目的

既存の組織論で個別に語られてきた以下の現象を、閉鎖集団の再生産構造として整理します。

- 組織閉鎖性
- 内部適応による外部不適応
- 外部知識の吸収不全
- 有害ルーティンの固定・伝播
- 組織の衰退・吸収・消滅

## 発想の流れ

```text
古人類の吸収仮説
→ 貴族内婚と家系の希釈
→ 小集団の疫学的脆弱性
→ 企業組織への転用
→ 組織再生産モデル
```

## 中核概念

- 組織内婚
- 吸収不妊
- 悪習感染
- 内部適応による外部不適応
- 閉鎖集団の自己希釈

## 現在の位置づけ

これは完成理論ではなく、理論プロトタイプです。まずは概念整理、既存理論との差分、応用可能性、反証条件を整理します。
